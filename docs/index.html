Keyboard Shortcuts

  • Y/ Space Confirm
  • H Toggle Help
  • C Navigate up in menu
  • V Navigate down in menu
  • S Switch to next tab
  • A Switch to previous tab
  • D New Tab
  • F Close Tab
  • X Back to Menu
  • G Toggle Old TV Effect

For combination shortcuts, hold the first key and press the second key as many times as needed.
